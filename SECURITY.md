# Security

qtBar is a World of Warcraft user-interface add-on. It runs only inside the game client and does not operate its own servers or handle real-world authentication.

## Reporting

If you believe you have found a security-relevant issue (for example, something that could harm users’ game clients or data in an unexpected way), please **do not** post full exploit details in public issues until it has been triaged.

**Preferred:** if this repository has [private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) or [Security Advisories](https://docs.github.com/en/code-security/security-advisories) enabled, use that. Otherwise open a **GitHub issue** with only a high-level description first; maintainers can follow up in the thread. We do not use email for security triage.

Include: affected version or commit, client region if relevant, and a short description of impact. We will acknowledge reasonable reports and coordinate any public disclosure.

## Expectations

We cannot guarantee a specific response time. For many UI add-ons, “security” issues are low severity; we will still take credible reports seriously.
